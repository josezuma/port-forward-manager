## Quick Start

```bash
git clone https://github.com/josezuma/port-forward-manager.git
cd port-forward-manager
python3 scripts/cli.py --help
```

## Demo

```bash
$ python3 scripts/cli.py
port-forward-manager v1.0.0
```

| Feature | Description |

| **Cross-platform** | Works on Linux, macOS, Windows |
